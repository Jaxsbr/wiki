What changed — and why

Typography: Fraunces + IBM Plex Sans instead of Inter Fraunces is an optical-size variable serif with personality — warm, editorial, immediately distinctive. IBM Plex Sans is neutral enough to stay out of the way. No gradient text, no all-caps labels before every heading.
Colour: warm cream/ink/amber instead of dark + purple gradient A deliberate light mode (rare among AI-generated sites). One accent colour (#d97706) applied with purpose — CTAs and emphasis only. No "VibeCode Purple".
Hero: left-aligned, specific copy, no orbs "Stop losing work to handoffs" is a specific claim about a real pain. Compare to "Build the Future — Without Limits". The hero visual is a structured grid pattern, not blurred decorative blobs.
Features: numbered horizontal bands instead of 3-col card grid Same information, completely different layout. No coloured left border, no glassmorphism, no floating icon above heading. The numbers create hierarchy without adding visual clutter.
Testimonial: one specific quote instead of a grid The testimonial names a specific, surprising thing that happened ("the QA sign-off never actually happened"). Generic praise ("it's a game-changer") doesn't build trust. Specificity does.
Border radius: deliberate variation, not uniform 16px Nav CTA has no border radius (sharp, authoritative). Feature tags have none. The primary button has none. Rounded corners appear only on supporting elements. The inconsistency is intentional — it creates visual rhythm.
